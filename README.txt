مشروع: نظام متابعة تربية الدواجن (Flask + SQLite)
اللغة: العربية (واجهة)
المزايا:
- إدارة دفعات (إضافة وعرض)
- تسجيل العلف، الأدوية، النمو اليومي
- لوحة معلومات Dashboard بسيطة
- تقرير لكل دفعة مع تقدير بيع حسب سعر الكيلو ومتوسط الوزن
تشغيل المشروع:
1. ثبت بايثون 3.8+
2. من المجلد root للمشروع نفّذ:
   pip install -r requirements.txt
3. شغّل التطبيق:
   python app.py
4. افتح المتصفح على: http://127.0.0.1:5000/
